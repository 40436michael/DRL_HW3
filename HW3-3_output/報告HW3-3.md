# 5. Experience Replay

為了降低資料相關性（correlation），使用 Replay Buffer：

```python
self.replay = deque(maxlen=1000)
```

每次訓練：

```python
minibatch = random.sample(
    self.replay,
    self.batch_size
)
```

隨機抽取 batch。

## Experience Replay 優點

- 打破 sequential correlation
- 提高 sample efficiency
- 增加訓練穩定性

---

# 6. Target Network

本次加入 Target Network：

```python
self.target_model = copy.deepcopy(
    self.model
)
```

並定期同步：

```python
if self.step_counter % self.sync_freq == 0:
    self.target_model.load_state_dict(
        self.model.state_dict()
    )
```

## 優點

避免：

```text
Q target 一直跟著 model 改變
```

造成 learning divergence。

---

# 7. Double DQN

本次作業實作 Double DQN：

```python
next_actions = torch.argmax(
    self.model(next_states),
    dim=1
)
```

使用：

- Main Network 選 action
- Target Network 評估 Q-value

```python
next_q_target = self.target_model(
    next_states
)

next_q = next_q_target.gather(
    1,
    next_actions.unsqueeze(1)
).squeeze()
```

## Double DQN 優點

傳統 DQN：

```text
容易高估 Q-value
```

Double DQN 可以：

- 減少 overestimation
- 提升學習穩定性
- 提高收斂品質

# 8. Gradient Clipping

本次加入：

```python
torch.nn.utils.clip_grad_norm_(
    self.model.parameters(),
    max_norm=1.0
)
```

## 目的

避免：

```text
Gradient Explosion
```

優點：

- 防止 loss 爆炸
- 提高訓練穩定性

---

# 9. Learning Rate Scheduler

本次使用：

```python
scheduler = optim.lr_scheduler.ExponentialLR(
    optimizer,
    gamma=0.999
)
```

每次更新：

```python
scheduler.step()
```

## 優點

隨著訓練進行：

- gradually decrease learning rate
- 提高後期收斂能力
- 避免震盪
