# 3. DQN（Deep Q-Network）介紹

DQN 為 Q-Learning 與 Deep Learning 的結合。

神經網路會輸入目前環境狀態（state），並輸出每個 action 的 Q-value。

Agent 會選擇 Q-value 最大的動作。

---

## Q-Learning 更新公式

[
Q(s,a)=r+\gamma \max_{a'}Q(s',a')
]

其中：

* ( s )：目前狀態
* ( a )：目前動作
* ( r )：reward
* ( \gamma )：discount factor
* ( s' )：下一個狀態

---

# 4. State Representation

GridWorld 的狀態會被轉換成：

```python
(4,4,4)
```

共四個 channel：

1. Player
2. Goal
3. Pit
4. Wall

之後 flatten 成：

```python
(1,64)
```

作為神經網路輸入。

此外，程式中加入少量 random noise：

```python
state = state + np.random.rand(1,64)/100.0
```

可以增加訓練穩定性。

# 6. Experience Replay

Experience Replay 為 DQN 的重要技術之一。

Replay Buffer 會儲存 Agent 與環境互動的經驗：

```python
(state, action, reward, next_state, done)
```

訓練時：

1. 將經驗存入 Replay Buffer
2. 隨機抽取 mini-batch
3. 更新 DQN 網路

---

## Experience Replay 優點

* 降低資料相關性
* 增加訓練穩定性
* 提高樣本利用率

---

## Replay Buffer 參數

```python
memory_size = 1000
batch_size = 64
```

# 7. Epsilon-Greedy 策略

本作業使用 epsilon-greedy 作為 exploration 方法。

Agent：

* 有機率隨機選 action
* 否則選擇最大 Q-value 的 action

---

## Epsilon 參數

```python
epsilon = 1.0
epsilon_min = 0.1
epsilon_decay = 0.995
```

初期大量探索（exploration），後期逐漸利用已學到的策略（exploitation）。
