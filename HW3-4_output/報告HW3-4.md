## 4.1 Double DQN

一般 DQN：

會使用同一個網路：

- 選 Action
- 評估 Q Value

容易產生：

- Q-value Overestimation

Double DQN：

將：

- Action Selection
- Action Evaluation

拆成：

- Main Network
- Target Network

程式碼：

```python
next_actions = torch.argmax(
    self.model(next_states),
    dim=1
)

next_q_target = self.target_model(
    next_states
)

next_q = next_q_target.gather(
    1,
    next_actions.unsqueeze(1)
).squeeze()
